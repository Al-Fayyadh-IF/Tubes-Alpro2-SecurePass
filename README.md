# SecurePass

Aplikasi sederhana untuk menyimpan dan mengelola data login akun secara lokal. Dibuat untuk Tugas Besar Algoritma Pemrograman 2 menggunakan bahasa Go.

## Cara menjalankan

Pastikan Go sudah terpasang, lalu jalankan:

```
go run main.go
```

## Fitur

- Tambah, ubah, dan hapus data akun
- Cari akun berdasarkan nama layanan (Sequential Search dan Binary Search)
- Urutkan akun berdasarkan nama (Selection Sort) dan waktu input (Insertion Sort)
- Statistik jumlah akun dan klasifikasi kekuatan password

## Data yang disimpan

Setiap akun menyimpan nama layanan, email, password, nama pengguna, dan tanggal update terakhir.

## Catatan

Input tidak boleh mengandung spasi. Fungsi clear screen memakai perintah `cls`, jadi dijalankan di Windows.
